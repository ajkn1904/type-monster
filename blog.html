A block is a piece of code inside a set of curly braces {} and groups code statements together. The difference between global scope & block scope is In JavaScript, a variable can be one of three different scopes, global, local, and block. The global variables can be accessed by any line of code in the program, including inside blocks while locally scoped ones are only visible within the function they are declared.

The event loop is a constantly running process that monitors both the callback queue and the call stack. If the Call Stack is empty, the Event Loop will take the first event from the queue and will push it to the Call Stack, which effectively runs it. Such an iteration is called a tick in the Event Loop.

There are 5 types of situation can be created when we get undefined values in javascript. They are: 1. The undefined is a primitive type that has a single value undefined. 2. Accessing an uninitialized variable returns undefined. 3. Accessing a non-existing property of an object returns undefined. 4. Accessing a out-of-bounds array element returns undefined. 5. A function without a return statement or with a return statement but without an expression returns undefined.
